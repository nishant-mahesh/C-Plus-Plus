Detailed Description

Greedy algorithm to find length of longest substring without repeating characters. The following resource was referred to.

We are given a string. We need to compute the length of the longest substring, such that no character in the substring occurs more than once.

For a string of length n, the time complexity of this algorithm is O(n) (amortized)
